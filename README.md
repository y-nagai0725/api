## 概要
* Javascriptのfetch機能によるapi連携の練習。
* localStorage機能を使用したユーザーの選択状態の保存、ダークモード切り替えの実装練習。
* 気象庁の天気予報apiを使用しています。

## 公開URL
[https://weather.mikanbako.jp/](https://weather.mikanbako.jp/)

## 目的
* 天気予報apiを使用してデータを取得、適切に加工して表示する。
